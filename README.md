# Mine Rush  -- **ALPHA**

## Description
Mine Rush is a mini game in minecraft where players get teleported to a random location with the objective on earning as many points as possible. Points can be obtained through mining almost any type of block and can be spent in the shop. Every 15 minutes the difficulty will increase making it more and more difficult to earn points where eventually the player will die and all points earned will be uploaded to the servers leaderboard and stats page!

## Dependencies

Skript 2.14.3
SkBee 3.18.0
